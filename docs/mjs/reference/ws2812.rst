``ws2812`` — Адресовані світлодіоди WS2812
------------------------------------------

Керування адресованими світлодіодами WS2812 ("NeoPixel") через периферію RMT.

Приклад:

.. code-block:: javascript
    :linenos:

    // Світимо 3 світлодіодами на піні 12: червоний, зелений, синій
    ws2812.write(12, [0xFF0000, 0x00FF00, 0x0000FF]);

Функції
^^^^^^^

.. js:function:: ws2812.write(pin, colors)

    Виводить кольори на стрічку світлодіодів WS2812.

    :param number pin: Номер піна даних (DIN).
    :param colors: Масив цілих чисел, кожне у форматі ``0xRRGGBB``.
